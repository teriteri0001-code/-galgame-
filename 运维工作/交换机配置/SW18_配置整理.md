## 四、AP 调试关键点 ⭐

### 物理链路检查
- **16 个 AP 口 (Gi1/0/1~16)** 全部配置了 `port access vlan 48` + `poe enable`
- 如果某些 AP 不通，优先检查：
  1. **PoE 是否供电正常** → `dis poe interface gi1/0/x`
  2. **端口是否 UP** → `dis interface gi1/0/x brief`
  3. **AP 是否获取到 IP** → 在核心上 `show ip dhcp binding` 查看 10.12.48.0/24 段
  4. **VLAN 48 到核心是否通** → 从 SW18 ping 10.12.48.1

### 流量路径
```
AP → SW18 (Gi1/0/1~16, VLAN 48) 
  → Po-channel1 (Gi1/0/18+20, Trunk) 
    → SW13 (Po1, Trunk) 
      → 核心 SW01 (VLAN 48 网关 10.12.48.1)
```

### 未使用的备用端口

- Gi1/0/25-28 完全未配置
